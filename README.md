# 🐝 Mochi de los Huertos — Learning Platform

Bilingual bee microlearning for families. Ages 6–8. English & Spanish.

## Live platform
→ Deploy via Vercel — connect this repo at [vercel.com/new](https://vercel.com/new)

## What's here
- `public/index.html` — home screen, pulls modules live from Supabase
- `public/storycard.html` — illustrated storycard player (offline-first + Supabase sync)
- `public/sw.js` — service worker for offline caching
- `vercel.json` — Vercel static deployment config

## Content pillars
1. 🍯 Bee hive life
2. 🗣️ Bilingual vocabulary (EN/ES)
3. 🌱 Permaculture & gardens
4. 🌰 Seed saving & biodiversity
